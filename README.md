### **👨‍💻 About Me**  
배움을 중요하게 생각하며, 개발을 통해 끊임없이 성장하는 것을 즐깁니다.  
최근에는 **크래프톤 정글(Krafton Jungle)** 부트캠프를 수료하고,  
**인터넷 강의, 스터디, 팀 프로젝트** 등을 병행하며 실무 역량을 키워가고 있습니다.
